## vmp
Minimalist CLI Music Player

<img src="https://raw.githubusercontent.com/vzze/vmp/main/screenshot.jpg">

Dependencies:
```
youtube-dl
ffmpeg
```

First time setup
```
Run build script, then the install script.
```

### Commands
* `download` [URL]
* `play queue` [Name] | `play` [range] | `play` all
* `list` Lists all of the installed songs 
* `queue create [Name]` Creates a new queue
* `queue remove [range]` Removes a range of songs from the queue
* `queue add [range]` Adds a range of songs to the queue
* `queue delete [Name]` Deletes the queue
* `queue list` Lists all of the created queues
* `queue peek` Lists all of the songs in a queue
